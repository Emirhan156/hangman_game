# Hangman Game

A simple console-based Hangman game written in C++. This is my first C++ project.

> Built without AI assistance - written entirely from scratch.

## How to Play
- The game randomly selects a word
- Guess the word letter by letter
- You have 4 wrong guesses before the game ends
- Press 1 to guess the full word
- Press 2 to play again
